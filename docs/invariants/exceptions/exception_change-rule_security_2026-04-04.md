# Exception: security structural change without invariants linkage

This exception was auto-generated by local preflight because a structural change was detected,
but invariants linkage (manifest/docs with INV-###) was not included in the same commit.

## Reason

Migration `0114_grant_onboarding_tables_to_app_role.sql` grants DML privileges on three
onboarding control-plane tables (`tenant_registry`, `programme_registry`,
`programme_policy_binding`) to `symphony_app_role`. This is required to unblock the
Pilot Demo "Seed Demo Tenant" workflow which was returning HTTP 503 due to missing
database permissions. The grant scope is intentionally narrow (SELECT, INSERT, UPDATE only —
no DELETE, no DDL) and follows the revoke-first privilege posture.
