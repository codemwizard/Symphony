# Glossary

| Term | Definition |
|------|------------|
| **ADR** | Architecture Decision Record |
| **Invariant** | A rule that must always hold; mechanically verified |
| **Migration** | SQL file that mutates schema; immutable once applied |
| **Baseline** | Derived schema snapshot for fresh DB convenience |
| **Outbox** | Transactional queue for reliable message dispatch |
| **Lease** | Time-bounded ownership claim on a work item |
| **Rail** | External payment network (SWIFT, RTGS, etc.) |
| **Policy** | Versioned rules governing transaction processing |
| **Participant** | Entity that sends/receives payments |
| **Tenant** | Logical isolation boundary for multi-tenancy |
